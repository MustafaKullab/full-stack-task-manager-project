# Full Stack Task Manager

A simple full-stack task management application built with Vue.js, Express.js, and MongoDB.

## Features

* User registration and login
* JWT authentication
* Password reset via email
* Avatar upload
* Create tasks
* Update tasks
* Delete tasks
* View personal tasks

## Tech Stack

### Frontend

* Vue 3
* Vite
* Bootstrap

### Backend

* Express.js
* MongoDB
* Mongoose
* JWT
* Multer
* Nodemailer

## Installation

### Backend

```bash
cd Backend
npm install
npm run dev
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

## Note

A `.env` file is required for backend configuration.
